elecV2P使用推送

  1. 2021-11-25
  2. 239字
  3. 2 min read

通知配置地址：v2p面板地址,setting菜单,通知相关

推送方式：

一、Telegram通知

通知链接：https://api.telegram.org/bot你的botapi/，选择 POST 方式

这里的botapi，其实就是你的bot_token，就是新建bot时，Botfather发给你的token。

示例，假如你的bot_token是12345678:NNVDISOEJNBNB1245，那通知链接如下填写：

https://api.telegram.org/bot12345678:NNVDISOEJNBNB1245/

如果使用TG反代的话，请使用cloudflare的连接

示例：https://xxxxx.xxx.workers.dev/botapi/

{
  "method": "sendMessage",
  "chat_id": 你的TG userid,
  "text": `$title$\n$body$\n$url$`
}

二、pushplus通知

pushplus官网：http://www.pushplus.plus

{
 "token": `xxx`,
 "title": `$title$`,
 "content": `$body$\n$url$`,
 "Content-Type": `application/json`
}

注：token在pushplus官网获得

三、server酱

server酱官网： http://sc.ftqq.com

通知链接： http://sc.ftqq.com/SCKEY.send， 选择 POST 方式

{
 "text": `$title$`,
 "desp": `$body$可以随便加点自定义文字[链接]($url$)`
}

注：sckey在官网获得

本文参考：https://www.kejiwanjia.com/jiaocheng/zheteng/notes/26176.html
