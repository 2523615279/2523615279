安装elecV2P

docker run --restart=always -d --name elecv2p -e TZ=Asia/Shanghai -p 80:80 -p 8001:8001 -p 8002:8002 elecv2/elecv2p

# 推荐使用命令
docker run --restart=always \
  -d --name elecv2p \
  -e TZ=Asia/Shanghai \
  -p 8100:80 -p 8101:8001 -p 8102:8002 \
  -v /elecv2p/JSFile:/usr/local/app/script/JSFile \
  -v /elecv2p/Lists:/usr/local/app/script/Lists \
  -v /elecv2p/Store:/usr/local/app/script/Store \
  -v /elecv2p/Shell:/usr/local/app/script/Shell \
  -v /elecv2p/rootCA:/usr/local/app/rootCA \
  -v /elecv2p/efss:/usr/local/app/efss \
  elecv2/elecv2p

# 升级 Docker 镜像。（如果没有使用持久化存储，升级后所有个人数据会丢失，请提前备份）
docker rm -f elecv2p           # 先删除旧的容器
docker pull elecv2/elecv2p     # 再拉取新的镜像
# 再使用之前的 docker run xxxx 命令重新启动一下

记得在服务器防火墙放行8100，8101，8102端口
搭建完成的elecV2P的访问地址在“自己的IP：8100”

配置elecV2P

一、开启ANYPROXY代理：在箭头位置双击就行了。

二、REWRITE：添加重写订阅地址

重写订阅地址：https://raw.githubusercontent.com/shaolin-kongfu/js_scripts/main/zq/zqcx.json

三、TASK：添加定时任务订阅地址

定时任务订阅地址：https://raw.githubusercontent.com/shaolin-kongfu/js_scripts/main/zq/zq_cron.json

四、添加MITM host：添加需要解析的域名，然后保存。

MITM host：kandian.wkandian.com

手机端配置

由于本人是穷逼，只有安卓机，所以本篇教程只介绍安卓怎么使用。

一、手机连接wifi

二、手机下载安装证书：访问 ip:8100下载

四、下载中青看点安卓版APP（不过好像最新的不能抓到时长body，所以在这里放个我能用的）

中青看点APP：链接:https://pan.baidu.com/s/1gcuVoWW0lnsFXTklSa6qfg ，提取码:b5w5
如果可以的话，请填入本人的邀请码：59458495

五、自动抓包：你的每一次点击，都会自动抓包相关数据同步到elecv2p的JSMANAGE，可以登录面板查看。

  • zq_cookie：打开app?我的?任务中心?抓取ck?登录elecv2p在JSMANAGE确认是否抓到
  • zq_timebody：打开app?任选一篇文章、视频阅读（要看完），等右下角那个圆圈转完，如何没抓到就多读几次。
  • zqwzbody：打开app，任选一篇文章，不用读，进去返回就行。（建议抓取150个文章body就行），抓取的视频body也是存在这个变量里面。

视频抓取必须点进个人主页，在哪看的视频才能抓到。（建议抓取60个就行）

  • zqqdbody：打开app，任务中心，点击那个签到领现金
  • zqlookStartbody：打开app，任务中心，如图所示，点击已完成退出即可抓到（任务中心下面那几个60个青豆的也属于看看赚，记得抓）
  • zqllzbody：打开app，任务中心，看看赚，浏览赚，点击去白拿，返回即可抓到
  • zqsszbody：打开app，任务中心，看看赚，搜索赚（只能抓到那里面带火图标的），点去搜索，然后随便搜索返回就行。
